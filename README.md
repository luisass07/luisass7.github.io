# Sistema ERP para Asaderos de Pollo

## Descripción del Proyecto

Este repositorio contiene la documentación de arquitectura del proyecto **Sistema ERP para Asaderos de Pollo**.

El objetivo es diseñar un sistema ERP modular basado en web que permita gestionar integralmente los procesos administrativos, operativos y financieros de los asaderos de pollo.

El sistema busca solucionar problemas comunes como:

- Control manual del inventario
- Falta de integración entre compras, ventas y contabilidad
- Errores en el cálculo de costos
- Gestión ineficiente de empleados
- Dificultad en el análisis financiero

---

## Módulos del Sistema

El ERP está compuesto por los siguientes módulos:

- Compras
- Facturación Electrónica
- Stock y Costos
- Activos Fijos
- Empleados
- EIS (Executive Information System)

---

## Arquitectura

El sistema está basado en:

- Aplicación web
- Arquitectura modular
- Base de datos relacional
- Integración contable automática
- Control de roles

## Objetivo

Diseñar un sistema ERP web especializado para asaderos de pollo, que permita gestionar de forma integrada los procesos de compras, facturación electrónica, control de inventario y costos, gestión de activos fijos, administración de empleados y análisis ejecutivo, con el fin de mejorar el control operativo, financiero y la toma de decisiones del negocio.
